# 飞书最小化插件修复版

基于飞书官方最小化架构的修复版本，专门解决以下问题：

## 修复的问题

1. **TypeError: (void 0) is not a function** - 修正了getFields方法调用错误
2. **CSP违规** - 避免使用data: URL的Web Worker
3. **配置获取权限错误** - 使用官方正确的API调用方式

## 核心特性

- ✅ 基于飞书官方最小化架构
- ✅ 使用官方正确的字段获取方法
- ✅ 避免CSP违规的代码实现
- ✅ 简化界面，只保留核心功能
- ✅ 专门处理防盗链图片下载
- ✅ 模拟谷歌浏览器请求头

## 使用方法

1. 部署到GitHub Pages或其他静态托管服务
2. 在飞书多维表格中添加自定义插件
3. 配置插件URL为部署地址
4. 输入URL字段名和附件字段名
5. 点击开始转换

## 技术实现

- 使用React + TypeScript
- 基于@lark-base-open/js-sdk官方SDK
- 实现智能图片下载和上传
- 支持批量记录处理

## 注意事项

- 确保有正确的API权限
- 字段名需要准确匹配
- 支持常见的图片格式
